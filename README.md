## 🚀 Run Locally

```bash
# Clone the repository
git clone https://github.com/majidawan01/Majid_awan_updated_portfolio.git

# Navigate into the folder
cd Majid_awan_updated_portfolio

# Install dependencies
npm install

# Start the development server
npm run dev
```
